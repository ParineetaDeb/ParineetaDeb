# SentinelsVault 🛡️

**SentinelsVault** is an enterprise-grade cryptographic password manager built with Python and SQLite3. It focuses on high-security data persistence and user-friendly credential management.

## 📄 Documentation
Detailed technical documentation for this project can be found below:

## 🛠️ Tech Stack
* **Language:** Python
* **Database:** SQLite3
* **Security:** Cryptography (AES-256, Argon2id, PBKDF2)
